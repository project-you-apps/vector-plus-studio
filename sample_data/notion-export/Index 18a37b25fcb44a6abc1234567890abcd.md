# Index

Tags: meta, navigation
Created: April 1, 2026
Aliases: Home, MOC

A small synthetic Notion export used to test the Cart Builder's
filename-normalization for Notion-flavored markdown. Same content as the
Obsidian vault sibling, but using Notion's URL-encoded + 32-char hash
filenames.

## People

- Galileo Galilei — telescope, Jovian moons, heliocentrism advocate
- Johannes Kepler — laws of planetary motion, optics, mysticism
- Tycho Brahe — most precise pre-telescope observations on record

## Concepts

- Heliocentrism — sun-centered model
- Geocentrism — earth-centered model
- Telescope — the instrument that broke the old model

## How this export is structured

Notion exports use markdown files with a 32-char hex hash suffix on
each filename and URL-encoded spaces. When dragged into the Cart
Builder, the source field should display the clean page name with the
hash removed.
